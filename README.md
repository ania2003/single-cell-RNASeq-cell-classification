# Data

The raw data are not included in this repository.

Before running the notebook, create a `raw` directory inside this folder and add authorised copies of:

```text
data/raw/single_cell.h5ad
data/raw/singlecell_train.csv
data/raw/random_singlecell.csv
```

Update the file paths in the notebook so that they point to these files.
